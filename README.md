# sottovoxbot

a telegram bot that allows you to send a private message to a certain user and no one except them will be able to read it.

you will need a redis instance running.

## installing

##### yarn

```sh
yarn
```

##### npm

```sh
npm install
```

## running

##### yarn

```sh
yarn dev
```

##### npm

```sh
npm run dev
```

## building

##### yarn

```sh
yarn build
```

##### npm

```sh
npm run build
```
